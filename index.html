Sutton Arts Member Database

PHP Codeigniter MySQL groceryCRUD jQuery

In 2013 I was approached by a local theatre looking to replace their Access database of Members with a system that could be centrally maintained and accessed by a number of administrative staff. After a little searching I came across an excellent codeigniter plugin called grocery CRUD.

The theatre is volunteer run, so the system allows them to search for members according to their interests - ie. acting, directing, front of house etc. Membeship is paid so the system keeps track of totals and includes a custom action to renew membership each year. Finally, the system includes the option to print labels to a PDF in order for the theatre to send out mailshots

Workdodger

jQuery Phaser

Workdodger is a simple arcade game created using a Javascript game engine called Phaser. Players need to guide their ship using the arrow keys and avoid the alien hordes for as long as possible

The game stores the top 10 high scores in a MySQL database using jQuery, so players can compete to get their name on the board!

Prediction Factory

PHP jQuery Bootstrap MySQL

The Prediction Factory is a dynamic web application running on a standard LAMP stack that allows users to predict the results of international football tournaments. The system scores the predictions and users can match their scores against their friends.

Users can join any number of leagues or create their own and invite their friends to sign up and make predictions

The application has a fully responsive front end created using Bootstrap
